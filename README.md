# Hot-Air-Control
Controls a large hot air balloon in Create: Aeronautics using ComputerCraft
# Instructions
Install Create, Create: Aeronautics, CC: Tweaked, and CC: Redstone Link Bridge, all from Modrinth
Place a computer in your airship
Place an altitude sensor on top of the computer (will be important for calibration when i add that)
Place a redstone link bridge on the right side of the computer
Run ```insert command here```
Run ```calibrate-airship```
Follow instructions in console
